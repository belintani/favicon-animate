favicon-animate - Animated Favicon Example

Animated Favicon Control

This page demonstrates animated favicon support with GIF/WebP images.

Badge Control

Add badges to your animated favicon:

Animation Settings

Status

Animation Status: Not started
Current Favicon: None
Badge Applied: No

Code Example

import { FaviconAnimator } from 'favicon-animate';

const animator = new FaviconAnimator({
  enableAnimation: true,
  pauseOnHidden: true
});

// Set animated favicon
await animator.setFavicon({
  url: '/path/to/animated.webp',
  animate: true,
  corsMode: 'cors',
  loopCount: 0
});

// Add badge
animator.setBadge({
  number: 5,
  position: 'top-right',
  backgroundColor: '#ff0000'
});

// Pause/Resume
animator.pause();
animator.resume();
